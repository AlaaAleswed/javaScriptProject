 HireGate

- Project Overview

HireGate is a web-based system used by companies to create and manage job assessment exams for new applicants.  
Users (candidates) can create an account, log in, take exams, and instantly receive their scores.  
Administrators have full control over exam management, including creating forms, editing and deleting them, and managing user accounts.

- Purpose of the Project

The system aims to streamline the hiring process by allowing companies to test applicants efficiently and automatically evaluate their performance.
